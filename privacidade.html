POLÍTICA DE PRIVACIDADE

Última atualização: abril de 2025 · Em conformidade com a LGPD (Lei nº 13.709/2018)

1. Controlador dos Dados

O responsável pelo tratamento dos seus dados pessoais é Dionatan Alves, atuando sob a marca TeamDiou, com contato via WhatsApp: (74) 99953-1223.

2. Dados Coletados

Coletamos os seguintes dados pessoais fornecidos voluntariamente por você ao preencher o formulário do site:

  • Nome completo
  • Endereço de e-mail
  • Número de WhatsApp
  • Dados da calculadora: sexo, objetivo, nível de experiência, idade, peso e altura
  • Dados de navegação: UTM parameters, URL de acesso, data e hora

Também coletamos dados de navegação de forma automática por meio de ferramentas de análise (Google Analytics 4, Microsoft Clarity e Google Tag Manager), incluindo endereço IP anonimizado, tipo de dispositivo, navegador e comportamento de navegação.

3. Finalidade do Tratamento

Seus dados são utilizados para as seguintes finalidades:

  • Execução do serviço: personalização do protocolo de treino e dieta conforme seus dados físicos
  • Comunicação: contato via WhatsApp para suporte e acompanhamento do programa
  • Marketing: envio de comunicações sobre o programa e lançamentos futuros (você pode cancelar a qualquer momento)
  • Análise e melhoria: compreender o comportamento de navegação para otimizar o site e as campanhas
  • Cumprimento legal: atender obrigações legais e regulatórias

4. Base Legal

O tratamento dos seus dados é fundamentado nas seguintes bases legais da LGPD:

  • Consentimento (Art. 7º, I): para envio de comunicações de marketing
  • Execução de contrato (Art. 7º, V): para entrega do programa adquirido
  • Legítimo interesse (Art. 7º, IX): para análise de navegação e melhoria dos serviços

5. Compartilhamento de Dados

Seus dados podem ser compartilhados com:

  • Google LLC — Google Analytics 4 e Google Tag Manager (análise de tráfego)
  • Microsoft Corporation — Microsoft Clarity (análise de comportamento)
  • Meta Platforms — API de Conversões para otimização de campanhas publicitárias
  • Infinity Pay — processadora de pagamentos (apenas dados necessários à transação)

Não vendemos, alugamos ou cedemos seus dados a terceiros para fins comerciais.

6. Cookies e Tecnologias de Rastreamento

Utilizamos cookies e tecnologias similares para:

  • Medir o desempenho das páginas (GA4)
  • Registrar sessões e mapas de calor (Clarity)
  • Atribuir conversões a campanhas de anúncios (Meta Pixel / CAPI)
  • Identificar a origem do acesso (UTM parameters)

Ao continuar navegando, você consente com o uso de cookies. Você pode desabilitá-los nas configurações do seu navegador, o que pode impactar a funcionalidade do site.

7. Retenção dos Dados

Mantemos seus dados pelo tempo necessário para cumprir as finalidades descritas nesta política ou conforme exigido por lei:

  • Dados de leads não convertidos: até 12 meses após a coleta
  • Dados de clientes: durante a vigência do programa e por até 5 anos para fins fiscais e legais
  • Dados de navegação (GA4, Clarity): conforme a política de retenção de cada plataforma

  • Corrigir dados incompletos, inexatos ou desatualizados
  • Solicitar a anonimização, bloqueio ou eliminação de dados desnecessários
  • Revogar o consentimento a qualquer momento
  • Solicitar a portabilidade dos dados

Para exercer qualquer um desses direitos, entre em contato via WhatsApp: (74) 99953-1223. Responderemos em até 15 dias.

9. Segurança

Adotamos medidas técnicas e organizacionais adequadas para proteger seus dados contra acesso não autorizado, perda, alteração ou divulgação indevida. Os dados transmitidos ao nosso formulário são enviados via HTTPS e processados com criptografia.
